# Literature and Daily Life 2.0

这是一个多人创作的项目,为游戏提供全新原创内容,从歌曲、新问题、到轻松有趣的主题讨论（全中文）
此项目使用了DDSP-SVC 6.2，项目地址:https://github.com/yxlllc/DDSP-SVC

# 核心

两百话题，以及两（？）首奖励曲目

将点进literature and Daily Life后的game复制粘贴到mas根目录即可
